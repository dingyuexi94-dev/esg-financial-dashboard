# ESG Financial Dashboard

An interactive dashboard to analyze ESG (Environmental, Social, Governance) performance and financial metrics.

## Features
- Interactive data visualization
- Real-time filtering
- Company performance analysis
- ESG scoring

## How to Run
1. Install R and RStudio
2. Run the following code in R:
```r
install.packages(c("shiny", "shinydashboard", "plotly", "dplyr", "DT"))
shiny::runApp("app.R")
